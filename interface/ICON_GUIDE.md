# NovaSector 图标机制说明（窗口左上角 vs 任务栏）

本文说明你刚遇到的问题：
- 已经改了 `skin.dmf`，窗口左上角图标变了
- 但 Windows 任务栏图标没变

这不是你改错了，而是 **两个图标来源不同**。

---

## 1. 窗口左上角图标（已生效）

这个图标来自 BYOND 皮肤文件 `interface/skin.dmf` 的窗口定义。

当前仓库已设置为：

- `interface/skin.dmf:68`（`window "mainwindow"`）
- `interface/skin.dmf:114`（`window "mapwindow"`）

两处都指向：

- `modular_nova\\master_files\\icons\\ui_icons\\common\\tianguan_logo.png`

这就是为什么你看到程序窗口左上角图标已经变成新图。

---

## 2. 任务栏图标（为什么没变）

Windows 任务栏图标通常不取 `skin.dmf` 的窗口 `icon`，而是取：

1. 启动进程本身（`dreamseeker.exe`）的应用图标，或
2. 你固定到任务栏的快捷方式 `.lnk` 的图标缓存

所以：

- 仅重编译 `.dmb` / 修改 `skin.dmf`，不会改变任务栏图标
- 这属于操作系统和启动器层，不是 DM 代码层

---

## 3. 你可以怎么改任务栏图标

### 方案 A（个人本机，最快）

做一个自定义快捷方式并固定到任务栏：

1. 准备 `tianguan_logo.ico`（建议 256x256 多尺寸 ico）
2. 新建快捷方式，目标指向 DreamSeeker 启动参数（你平时进服的方式）
3. 右键快捷方式 -> 属性 -> 更改图标 -> 选择 `tianguan_logo.ico`
4. 用这个快捷方式启动一次
5. 从任务栏取消旧图标固定，再固定新启动项

> 这是最稳妥的“任务栏换图标”方式。

### 方案 B（面向所有玩家）

制作自定义启动器（exe）并替换其应用图标，然后让玩家用该启动器进服。

- 优点：全员任务栏图标统一
- 缺点：需要额外维护启动器

---

## 4. 常见误区

- 误区：`skin.dmf` 改了就一定会改任务栏图标
  - 实际：通常只改 BYOND 窗口图标，不改任务栏应用图标

- 误区：重编译 `.dmb` 可以刷新任务栏图标
  - 实际：任务栏常由 Windows 图标缓存/快捷方式决定

---

## 5. 故障排查（任务栏仍不变）

按顺序检查：

1. 是否从“新快捷方式”启动，而不是旧固定项
2. 是否取消固定了旧任务栏图标后重新固定
3. 是否彻底关闭 DreamSeeker 进程后再开
4. 是否仍引用旧 `.ico` 路径或不存在的 `.ico`

---

## 6. 结论

- DM 仓库里可控的是 `skin.dmf` 的窗口图标（你已经改成功）
- Windows 任务栏图标要通过快捷方式/启动器层解决
- 你的现象是正常机制表现，不是修改失败
